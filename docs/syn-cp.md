# syn-cp

Interact with the Synadia Control Plane

```
syn-cp [flags]
```

**Options**

```
  -h, --help            help for syn-cp
      --server string   Synadia Control Plane Server
      --token string    Synadia Control Plane Auth Token
  -v, --version         version for syn-cp
```
